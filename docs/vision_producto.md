# Visión del producto

Empleos Gastronómicos será una plataforma web enfocada exclusivamente en el rubro gastronómico.

## Objetivo
Permitir que restaurantes, cafeterías, hoteles, bares, dark kitchens y negocios gastronómicos publiquen ofertas laborales, y que los candidatos puedan postular de forma simple, rápida y profesional.

## Inspiración
La plataforma tendrá una lógica similar a Get on Board, pero especializada en gastronomía.

## Funciones principales iniciales
- Registro e inicio de sesión
- Perfil de empresa
- Perfil de candidato
- Publicación de ofertas laborales
- Postulación a empleos
- Filtros de búsqueda
- Panel de administración
- Sistema de valoraciones
- Base preparada para importar ofertas externas

## Stack técnico
- PHP
- MySQL
- Bootstrap 5
- JavaScript vanilla
- Arquitectura MVC simple
